# Project Roadmap

This document outlines the development roadmap for the **Indoor Navigation System**. It is a living document and will be updated as we progress.

## 🏆 Goals for 2026

- **Q1**: Core Infrastructure & Basic Navigation (Completed ✅)
- **Q2**: Accessibility & Voice Guidance
- **Q3**: Advanced Navigation Features & Optimization
- **Q4**: Mobile-Specific Hardware Integration (AR/Bluetooth)

---

## 📅 Timeline

### Phase 1: Core Foundation (Current v1.0.0)
- [x] Setting up Clean Architecture structure
- [x] Firebase integration (Auth & Firestore)
- [x] Admin panel for map data entry
- [x] A* Pathfinding algorithm implementation
- [x] Basic user wayfinding UI

### Phase 2: Accessibility & Usability (Next - v1.1.0)
- [ ] **Voice Guidance**: Text-to-speech navigation instructions.
- [ ] **High Contrast Mode**: Toggle for visually impaired users.
- [ ] **Screen Reader Optimization**: Semantics and labels for TalkBack/VoiceOver.
- [ ] **Advanced Accessibility Routing**: Full wheelchair support (ramps, elevators only).
- [ ] **Multi-Language Support**: Localization for key campus languages.

### Phase 3: Advanced Navigation (v1.2.0)
- [ ] **Multi-Destination Routing**: Plan a route with intermediate stops.
- [ ] **Point of Interest (POI) Search**: Find verify-specific amenities (coffee, restrooms, printing).
- [ ] **ETA Calculation**: Estimate travel time based on walking speed.
- [ ] **Crowdsourced Path Updates**: Allow users to report closed paths or obstacles.
- [ ] **Offline Mode**: Cache map data for navigation without internet.

### Phase 4: Hardware Integration (v1.3.0)
- [ ] **Indoor Positioning System (IPS)**: Use Bluetooth Low Energy (BLE) beacons for real-time location.
- [ ] **AR Navigation**: Augmented Reality overlay for camera-based wayfinding.
- [ ] **QR Code Scanning**: Scan codes at room entrances to set "Current Location".
- [ ] **Haptic Feedback**: Vibration patterns for turns and arrival.

---

## 🔮 Future Ideas (Backlog)

- **Event Navigation**: "Take me to the Career Fair" deep linking.
- **Parking Integration**: Navigate from parking lot to building entrance.
- **Emergency Mode**: Quickest exit routes during fire drills.
- **Web Admin Dashboard**: Dedicated React/Angular web app for map management.
