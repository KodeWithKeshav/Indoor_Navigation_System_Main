# User Manual

Welcome to the **Indoor Navigation System**! This manual will guide you through using the application to navigate your campus efficiently.

## 🗝 Getting Started

### 1. Login
- **Launch the App**: You will see the login screen.
- **Student/Staff**: Enter your university email and password.
- **Guest**: Tap "Continue as Guest" for limited access (navigation only).
- **Admin**: Login with administrator credentials to access map management tools. (Logout prompts a confirmation dialog to prevent accidental session loss).

### 2. Home Screen
- **Search Bar**: Tap to find a room quickly.
- **Map View**: Interactive map of the current floor.
- **Floor Switcher**: Floating buttons to change floors (e.g., G, 1, 2).
- **Recent Locations**: Quick tap access to places you've visited before.

## 🧭 Navigation

### Planning a Trip
1. Tap the **"Where to?"** search bar.
2. **Select Start Location**: Defaults to "Current Location" (if available), or type a room name (e.g., "Library").
3. **Select Destination**: Type the room name or number (e.g., "Lab 301").
4. **Browse Mode**: Tap the folder icon to browse by Building > Floor > Room.
5. **Add Waypoints (Errands)**: Tap "**Add Stop**" below the destination to add multiple locations to your trip. The system will automatically optimize the visit order to minimize total travel time!

### Customize Route
- **Accessible Route**: Toggle the wheelchair icon ♿ to avoid stairs and use elevators.
- **Swap**: Tap the ⇅ arrows to reverse start and end points.

### During Navigation
- **Step-by-Step**: Follow the cards at the top of the screen.
- **Visual Cues**:
  - 🟢 **Green Arrow**: Go straight.
  - ↩️ **Turn Left/Right**: Follow the turn.
  - 🪜 **Stairs**: Go up/down stairs.
  - 🛗 **Elevator**: Use the elevator to change floors.
- **Arrival**: A checkered flag 🏁 indicates you have reached your destination.

## 🛠 Admin Features (Admins Only)

### Managing Map Data
Access the **Admin Panel** from the side drawer.

#### Adding a Building
1. Go to **Organizations** > Select Campus.
2. Tap **+ Add Building**.
3. Enter Name, Code, and number of floors.

#### Editing Floor Plans
1. Select a building > Select a floor.
2. **Add Room**: Tap a location on the grid/map. Enter room details.
3. **Add Connection**: Select "Corridor", "Stair", or "Elevator" tool. Draw connections between rooms.
4. **Toggle Out-of-Service**: Edit an existing room to mark it "Out of Service". It will immediately turn red on the grid and be excluded from all user navigation routes.

## ❓ FAQ

**Q: Why can't I see the map?**
A: Ensure you have an active internet connection to load map data for the first time.

**Q: The app says "No route found".**
A: The destination might be disconnected from your starting point (e.g., under construction). Try selecting a different nearby room.

**Q: How do I report a map error?**
A: Use the "Report Issue" button in the settings menu.
